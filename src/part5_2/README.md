# part5_2. Model Quantization & Performance Profiling

## Task: 
Compare the performance of your chosen model across two different quantization levels (e.g., 1-bit vs 4-bit vs 8-bit).

Report on tokens per second (TPS) and peak VRAM/RAM usage.

## Analysis: 
Provide a summary of the trade-off between speed and output quality.

A performance report (table) detailing Tokens Per Second (TPS) and Peak VRAM/RAM usage for both versions.
